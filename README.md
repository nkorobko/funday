# FunDay 🎉

A fun-day planning tool for Employee Experience managers: pick an activity type, city, budget, and group size — get matching activities from a curated catalog with the cost math done for you.

- **Fully static** — hosted on GitHub Pages, no backend, no secrets, $0/month.
- **Hebrew-first, RTL, mobile-friendly.**
- Catalog lives in `data/activities.json`; the repo is the database.
